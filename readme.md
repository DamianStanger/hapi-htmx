# README
This is a demo app showing how to use html over the wire to achieve a responsive server side
rendered single page like app. 

## Technology
* https://hapi.dev/
* https://hapi.dev/module/vision/
* https://handlebarsjs.com/
* https://htmx.org/
* https://nodemon.io/
